Jogo dos Países
Autor: Thiago Sérvulo Guimarães
Data:28/08/2021

Este jogo foi desenvolvido em linguagem Python, com auxílio do QtDesigner para a criação da interface gráfica.
O objetivo do jogo é que o jogador adivinhe a que país pertence as bandeiras que são mostradas.
Estas bandeiras são carregadas via requisição HTTP do Google Imagens.
Para deixar a ordem das bandeiras de forma aleatória, um país é sorteado dentro de uma lista contendo todos os países.
Caso o jogador não saiba de qual país é a bandeira mostrada ele pode usar uma das opções de ajuda.
A primeira é o botão Pula, onde ele pula a bandeira atual.
A segunda é o botão Meio a Meio, onde duas opções são eliminidas.
A terceira é o botão Dica, onde ele recebe uma dica de qual continente se encontra o país ou qual é a sua capital.
É válido ressaltar que as opções de ajuda só podem ser usadas uma vez cada durante o jogo.
